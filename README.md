# ZackEyes — Public Release Channel

DMG downloads for [ZackEyes](https://github.com/yangshiqi/ZackEyes), a macOS Dynamic-Island companion for Claude Code.

See the [Releases](https://github.com/yangshiqi/ZackEyes-release/releases) tab for installers.

The in-app update checker polls this repo's `/releases/latest` endpoint and downloads the DMG asset directly — no GitHub authentication required.
