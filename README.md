# ZackEyes

ZackEyes is a macOS notch command center for AI coding agents. It keeps Claude Code and Codex CLI activity visible at the top edge of the screen, so you can stay in your editor while still seeing session state, approval prompts, and progress signals.

## Download

- Latest installer: [ZackEyes 0.4.2](https://github.com/yangshiqi/ZackEyes-release/releases/download/v0.4.2/ZackEyes-0.4.2.dmg)
- All installers: [Releases](https://github.com/yangshiqi/ZackEyes-release/releases)

## Features

- **Dynamic Island for coding agents**: shows agent state, running tasks, and pending attention in a compact macOS notch surface.
- **Claude Code and Codex CLI companion**: designed for local AI coding workflows that need lightweight status visibility outside the terminal.
- **Permission and action awareness**: surfaces moments where the agent needs user input, approval, or follow-up.
- **Release-friendly update channel**: the app can check this repository's latest release and download the DMG asset directly.
- **Native maCOS experience**: packaged as a DMG installer and optimized for a quiet desktop workflow.

## Feedback and Requests

Use [Issues](https://github.com/yangshiqi/ZackEyes-release/issues) as the public feedback channel.

Please open an issue for:

- Bugs, crashes, install failures, or update problems.
- Feature requests and workflow ideas.
- Compatibility feedback for macOS versions, Apple Silicon / Intel devices, Claude Code, or Codex CLI.
- UI, copy, icon, or interaction suggestions.

When reporting a problem, include:

- ZackEyes version.
- macOS version and Mac model.
- Which agent or CLI you were using.
- Steps to reproduce the issue.
- Screenshots or logs when they help explain the behavior.

## Update Channel

The in-app update checker polls this repository's `/releases/latest` endpoint and downloads the DMG asset directly. No GitHub authentication is required for users.
